# Aktivitetsväljare

## Syfte
Hjälper användaren att välja en aktivitet baserat på deras preferenser och för att kunna ge snabb beslut om vad man kan göra.  
Dvs att Användaren väljer antal personer, tid och energinivå, och får direkt ett förslag på en aktivitet.

---

## Funktioner
Användaren kan:
- Välja antal personer (1, 2–4 eller 5+)
- Välja tillgänglig tid (kort, medel eller lång)
- Välja energinivå (låg, medel eller hög)
- Klicka på en knapp för att få ett aktivitetsförslag
- Se resultatet direkt efter på sidan

---

## Teknik
Projektet använder:
- HTML - struktur
- CSS - design
- JavaScript (logik och interaktivitet)
- Google Fonts - externta bibliotek för typsnitt - Poppins
